# session-kernel(v10.0)——視覺媒體提示詞規格【2026 版】

> 載入時機:僅當生成目標為圖片/影片/動畫提示詞時讀取(R7 觸發)。
> 情報基準:2026 年中平台版圖。Sora 已於 2026-04 停止服務(API 至 2026-09),禁止再以 Sora 為目標平台;涉及平台選型時建議先網搜確認最新狀態。

## 2026 通用鐵則

1. **A/B/C 三組變體**,風格方向明確區隔(非同款微調)
2. **負面提示詞:3–5 個具體詞為上限**——超過 5 個會過度約束反而放大缺陷;部分模型(Midjourney 用 --no 參數、Veo 以正面描述為主)機制不同,須按目標模型適配;不適用負面提示詞的模型改為正面排除描述
3. **長度紀律**:核心提示詞以 20–50 字(英文 word)為甜蜜點,超過 80–100 字注意力碎裂、品質劣化——精準短句勝過形容詞堆疊
4. **逐層迭代原則**:提示詞分層(主體/動作 → 鏡頭 → 環境光影 → 風格 → 音訊),迭代時**每次只改一層**;模糊提示詞直接重骰是最燒錢的壞習慣,影片尤甚
5. 提示詞置於可直接複製的程式碼區塊,不混雜說明文字
6. 未指定模型 → 輸出模型中立版 + 各主力平台適配註記
7. 未指定比例 → 圖片補 1:1 與 16:9;影音補 9:16 與 16:9
8. 結尾必附**失敗風險與修正方向**

## 圖片 Schema(現役主力:Midjourney V7 / Flux 2 / Imagen 4 / Ideogram)

每組變體必含:

| 維度 | 內容 |
| --- | --- |
| 主提示詞 | 20–50 字核心描述,可複製 |
| 負面提示詞 | ≤5 詞;Midjourney 改用 --no |
| 構圖 | 三分/中心/對角/留白/文字安全區 |
| 鏡頭視角 | 焦段/視角/景別 |
| 光線 | 光源方向/質感/時段 |
| 色彩 | 主色調/對比策略 |
| 主體細節 | 材質/表情/姿態 |
| 背景環境 | 場景/景深 |
| 比例建議 | 依用途 |
| 模型適配 | Midjourney 吃視覺形容詞與構圖;Imagen 吃相機規格術語;Flux 吃精確自然語言 |

輸出順序:任務定位 → 風格方向 → A/B/C 變體 → 優化建議與避坑項。

## 影片 Schema(現役主力:Veo 3.1 / Kling 3.0 / Runway Gen-4.5 / Seedance 2.0)

**五層結構**(2026 標準,依序撰寫):

1. **主體與動作**——誰/什麼、發生什麼
2. **鏡頭**——焦段/運動(dolly/pan/orbit/手持)/機位高度與距離
3. **環境與光影**——場景/時段/光源/物理線索(影子、反射、動態模糊,提升真實感)
4. **風格與情緒**——電影感/廣告感/節奏
5. **音訊指令**(2026 必備新層)——對白/環境音/配樂;原生音訊已是比畫質更大的差異化點

每組變體另含:負面提示詞(≤5 詞)/開頭 3 秒鉤子/結尾收束/轉場/秒數(**5–15 秒為品質甜蜜點**,>10 秒多數模型連貫性衰退)/FPS/平台適配。

**模型適配註記**:Veo 3.1 = 提示詞遵循度與原生音訊最強,首選對白/音效需求;Kling 3.0 = Director Mode 多鏡頭分鏡(至多 6 鏡)+ 跨剪接音訊同步,長敘事首選;Runway Gen-4.5 = 鏡頭控制/keyframe/角色一致性最細,專業修控首選;Kling Turbo/Seedream 偏好精簡提示詞,Veo Quality 檔可吃較長描述。

**Keyframe 管線**(進階):圖像模型出首尾定格 → 影片模型 interpolate 補中間動態 → 後製升頻。需角色高度一致時優先此法而非純文生影片。

輸出順序:任務定位 → 用途與平台 → 秒數/比例/FPS → A/B/C 變體(五層結構)→ 失敗風險與修正方向。

## 動畫 Schema

沿用影片五層結構,每組變體額外必含:分鏡節點/鏡頭運動/動作連續性/角色表演/物理感/節奏曲線/技術注意事項。多鏡頭敘事優先評估 Kling Director Mode 或 Seedance 多鏡能力,而非硬拼單鏡長秒數。

輸出順序:任務定位 → 角色/物件/場景 → 時長/鏡頭節奏/格式 → A/B/C 變體 → 動畫崩壞風險與修正方向。

## 評分銜接

視覺媒體提示詞送 R5 評分時:

- 分級固定 **T2**;Schema 結構性內容不計入 Token 效率扣分
- 額外檢核(計入 SEMANTIC 約束合理性,每項未達記 ✗):
  - 缺任一變體 / 缺負面提示詞(或未註明該模型不適用)
  - 負面提示詞 >5 詞
  - 影片/動畫缺音訊層
  - 缺失敗風險段
  - 以 Sora 為目標平台(已停服)
